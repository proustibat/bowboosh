bowboosh

Simple boilerplate with gulp, bower, bootstrap sass, font-awesome, bootswatch themes

Quick usage

npm install
npm help

Customization

Edit ./config.json

Bootstrap theme could be one of bootswatch theme :

  • cerulean
  • cosmo
  • cyborg
  • darkly
  • flatly
  • journal
  • lumen
  • paper
  • readable
  • sandstone
  • simplex
  • superhero
  • united
  • yeti

Tests

npm install
npm test

Dependencies

  • bower: The browser package manager
  • gulp: The streaming build system

Dev Dependencies

  • gulp-bower: Install Bower packages.
  • gulp-concat: Concatenates files
  • gulp-filter: Filter files in a vinyl stream
  • gulp-help: Adds a default help task to gulp and provides the ability to add custom help messages to your gulp tasks
  • gulp-load-plugins: Automatically load any gulp plugins in your package.json
  • gulp-load-subtasks: A globular loader for gulp subtasks
  • gulp-notify: gulp plugin to send messages based on Vinyl Files or Errors to Mac OS X, Linux or Windows using the node-notifier module. Fallbacks to Growl or simply logging
  • gulp-sass: Gulp plugin for sass
  • gulp-uglify: Minify files with UglifyJS.
  • main-bower-files: Get main files from your installed bower packages.
